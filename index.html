À propos de GitHub

GitHub est une plateforme web pour le contrôle de version et la collaboration, construite sur Git. Elle permet aux développeurs d’héberger, de réviser et de gérer du code, ainsi que de suivre les problèmes et de travailler ensemble sur des projets depuis n’importe où.

  • Hébergement de code : GitHub fournit des dépôts cloud pour stocker et partager du code.
  • Collaboration : Plusieurs personnes peuvent travailler ensemble sur des projets, relire du code et proposer des modifications via des pull requests.
  • Suivi des problèmes : GitHub offre des outils pour suivre les bugs, les améliorations et les tâches du projet.
  • Communauté open source : Des millions de projets open source sont hébergés sur GitHub, en faisant un centre pour les développeurs du monde entier.
  • Actions & automatisation : GitHub Actions permet d’automatiser des workflows comme les tests et le déploiement.

Lancé en 2008, GitHub est devenu le plus grand hébergeur de code au monde et est largement utilisé par des particuliers, des équipes et des organisations.
